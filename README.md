# H-Clinic 🏥 | Hospital Management System

A modern, full-stack Hospital Management application built with **Django**. This platform allows patients to browse specialist doctors, book appointments, and manage their healthcare schedule through an intuitive, "Glassmorphism" inspired dashboard.

---

## 🚀 Key Features
- **User Authentication:** Secure signup and login system for patients.
- **Dynamic Dashboard:** A personalized hub featuring "Health Tips" and quick-access cards.
- **Appointment Booking:** Real-time scheduling with specialized doctors.
- **Status Tracking:** Visual indicators (Confirmed/Pending) for all scheduled visits.
- **Modern UI/UX:** Responsive design featuring glassmorphism effects and professional medical aesthetics.

---

## 🛠️ Tech Stack
- **Backend:** Python 3.x, Django Framework
- **Frontend:** HTML5, CSS3 (Custom Glassmorphism & UI components), Milligram CSS
- **Database:** SQLite (Relational mapping for Users, Doctors, and Bookings)
